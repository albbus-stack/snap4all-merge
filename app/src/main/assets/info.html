How it works

To finalize the installation of Snap4all the device must be restarted. Once restarted, the new features will be shown in the start screen. Without restarting you can only use the Termux console or read this info page. If the Termux console is disabled please check your internet connection, then close and reopen the app before restarting.
There are three main section of this app:

  1. Termux console: this is the Termux console of your device (since you cannot install this app and the official Termux app at the same time). Do your own research and use at your own risk.
  2. Iot app editor: opens the iot app editor of your local Node-RED server where you can build your own iot app.
  3. Iot app dashboard: opens the iot app dashboard of your local server where you can monitor your own iot app after having configured it properly following the Node-RED documentation. You can also use this to display the contents of a node-red-dashboard app.

Obviously the iot app editor and dashboard buttons are enabled only after the Node-RED server starts up.
* is possible that a toast stating "starting node-red" is shown several times, this is normal since the operating system closes the server following Android's battery-saving policy, the termux-boot service will try to restart it as soon as it is closed.

Example usage

  1. Install the application.
  2. Reboot your phone.
  3. Open the application.
  4. Open the iot app editor and enjoy the full Node-RED framework on your phone.
